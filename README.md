# Full Stack Open — Part 1 (monorepo)

This repository contains the three Part 1 exercise apps as separate Vite projects:

- `courseinfo/` — exercises 1.1–1.5
- `unicafe/` — exercises 1.6–1.11
- `anecdotes/` — exercises 1.12–1.14

## Run locally

From each folder:

```bash
npm install
npm run dev
```
